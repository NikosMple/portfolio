## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- Framer Motion
- React Icons

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```
